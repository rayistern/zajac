# Merkos Rambam — Frontend Mockup Spec

## Overview

Mobile-first PWA. Spotify-inspired daily Torah learning experience. Dark/moody theme.

---

## Player View (Main Experience)

- **Hero image:** On first load, fills 60–70% of the viewport. As the user scrolls down, the hero collapses and the content feed begins.
- **Content feed:** A vertical scroll of interleaved content blocks — halacha text, illustrations, charts, infographics, sichos. Content types flow naturally into each other with zero friction; the user shouldn't have to do anything to go from image to chart — it just appears as the class reaches that point.
- **Halacha text:** Paragraph-length blocks (not single lines). Vertical scroll.
- **Audio controls:** Fixed mini-bar at bottom (like Spotify).
- **Raise Hand:** Button that triggers voice conversation with AI.

## Other Screens

- Home / content discovery
- Raise Hand / voice conversation
- Perek detail / pre-play
- Progress & streaks
