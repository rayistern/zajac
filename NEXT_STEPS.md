## Current State

All 10 increments (0–9) are complete. The project has:

- **API:** 6 Hono routes (content, rambam, sichos, share, preferences, webhook), Drizzle ORM, seed script
- **Frontend:** React + Vite PWA with 15 components, skeleton loaders, error/empty states, a11y, dark theme
- **Pipeline:** 13 Python modules (RSS ingester, dual transcriber, 4-pass LLM aligner, artifact planner, image generator, Telegram voter, WhatsApp sender, reconciliation, orchestrator), 18 SQLAlchemy tables, Alembic
- **Infra:** AWS CDK (VPC, RDS, ECS Fargate, S3 content bucket, CloudFront with content-assets behavior)
- **Observability:** PostHog analytics, Sentry error tracking (both no-op without env vars)
- **PWA:** Workbox caching (API content, Rambam text, sichos, images, fonts), installable manifest

Everything builds clean. Pipeline imports verified.

## What's Needed to Go Live

### 1. AWS Infrastructure Setup
```bash
# Deploy CDK stack (requires AWS credentials configured)
cd infra && npx cdk deploy --context environmentName=staging
```

Required GitHub Actions variables:
- `AWS_REGION`, `DOMAIN_NAME`, `CLOUDFRONT_CERT_ARN`
- AWS OIDC trust policy for the repo

### 2. Configure Secrets

**Pipeline (env vars or AWS Secrets Manager):**
```
DATABASE_URL=postgresql://...
SOFER_AI_API_KEY=...
OPENAI_API_KEY=...
ANTHROPIC_API_KEY=...
REPLICATE_API_TOKEN=...
TELEGRAM_BOT_TOKEN=...
TELEGRAM_VOTING_CHAT_ID=...
```

**API (ECS task definition):**
```
DB_HOST, DB_NAME, DB_USER, DB_PASS  (from RDS secret)
SENTRY_DSN=...
```

**Frontend (build-time):**
```
VITE_POSTHOG_KEY=...
VITE_POSTHOG_HOST=https://us.i.posthog.com
VITE_SENTRY_DSN=...
```

**WhatsApp (API + Pipeline):**
```
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
WHATSAPP_PHONE_ID=...
```

### 3. Register a Class + Run Pipeline
```bash
# Insert a class record into the database
# Then run the pipeline:
cd pipeline
python -m src.pipeline.main run --stages ingest transcription sefaria alignment planning generation
```

### 4. Seed Editorial Reviewers
- Create Telegram group for voting
- Set TELEGRAM_VOTING_CHAT_ID to that group
- Recruit 6–10 volunteers

### 5. Deploy to Staging
```bash
git checkout -b test
git merge main
git push origin test  # triggers deploy-staging.yaml
```

### 6. Deploy to Production
```bash
git checkout -b production
git merge test
git push origin production  # triggers deploy-production.yaml
```
